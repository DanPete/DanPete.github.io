Shopify Contest App

Shopify app that lets you run contests based upon orders within a specific time range. The app can also, optionally, add a condition to limit contests based on the total order price. This project utilizes the Shopify API to pull all orders and their information from the Shopify shop. The app was created in Ruby on Rails using the Shopify App Gem and has basic use of React for front-end views.

  • Date: March 2016 - Ongoing Development
  • Code: Ruby on Rails, Shopify API, React, Bootstrap, PostgreSQL

Zoom Photo Gallery

Photo gallery for a Shopify stores that changes the featured image when clicking on variant images. The images are capable of zooming on hover with ElevateZoom. The gallery is also mobile responsive by converting to a image slider that has touch capabilities to zoom and pan on images as well.

  • Live Demo: Zoom Photo Gallery
  • Date: March 2017
  • Code: JavaScript, jQuery, Liquid, Shopify Themes, Hammer.JS, ElevateZoom

JavaScript Etch-a-Sketch

JavaScript and jQuery application to mimic Etch-a-Sketch using properties to transform the CSS by size and color. Dynamically changes multiple variable to populate different versions and styles of the Etch-a-Sketch.

  • Live Demo: JavaScript Etch-a-Sketch
  • Date: November 2016
  • Code: JavaScript, jQuery, CSS, HTML

Social Media Clone

Following the Michael Hartls' Ruby on Rail Tutorial, this project is a clone of the features and style of many social media website. Using REST architecture and MVC framework, the clone allows users to create an account and interact with other users using the platform.

  • Live Demo: Social Media Clone
  • Date: October 2016
  • Code: Ruby on Rails, SQLite, ImageMagick, Minitest, Guard, Bootstrap
